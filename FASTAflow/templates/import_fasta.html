Upload Your FASTA File

Welcome to this FASTA file analysis tool! Upload your FASTA file with genetic sequences and start a simple analysis.

What can you upload

For this application to work only the FASTA file extension is allowed. Those are the following extensions: '.fasta', '.fas', '.fa', '.fna', '.ffn', '.faa', '.mpfa', '.frn'. Any file uploaded with a different extension will be blocked.

To get started, simply choose your FASTA file and click the "Upload and Analyze" button. Both single and multiple sequence FASTA files are allowed.
